## APIs
- **Airship Push API** — Send push notifications across iOS, Android and web channels.
- **Airship Channels & Named Users API** — Register, list and manage channels (devices) and named users.
- **Airship Segments API** — Create and manage segments and tags for targeting.
- **Airship Automation & Journeys API** — Build automated and triggered journeys across channels.
- **Airship Email & SMS API** — Send email and SMS messages alongside push.
- **Airship Reports API** — Read message and engagement reports.
